Requirements
------------

-   Python >= 3.10

Setting Up the Environment
--------------------------

This site uses a pretty standard Python project environment:

```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Starting the Site
-----------------

```
mkdocs serve
```

Visualization
-------------

```
cd viz/
npm install
npm run
```
